# Unit Converter

A PowerToys Command Palette extension that converts between inches and millimeters. Designed for mechanical engineers who frequently switch between imperial and metric units.

## Capabilities

- Decimal values with unit: `25.4 mm`, `1 in`, `100mm`
- Fractions (inches): `3/8`, `3/8 in`
- Mixed fractions (inches): `1 3/4`, `2 1/2 in`
- Sheet metal gauge lookup: `18g`, `18 ga`, `18 gauge`
- Bare number: shows both directions simultaneously

Each result shows a rounded value and an exact value. Pressing Enter copies the result to the clipboard.

## Requirements

- Windows 10 19041 or later
- PowerToys 0.87 or later with Command Palette enabled
